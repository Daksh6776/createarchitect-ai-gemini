# CreateArchitect Groq + Gemini

This is a CreateArchitect AI server and UI that uses **Groq (LLaMA 3.1)** for main chat
and **Gemini 1.5 Flash** for schematic/blueprint JSON generation.

- Backend: Node.js + Express
- Frontend: ChatGPT-style HTML/CSS/JS
- Models: Groq (chat) + Gemini Flash (blueprints)
